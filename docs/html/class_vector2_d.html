Detailed Description

VECTOR 2D OVERVIEW

This class is a mathematical vector in a 2D Cartesian coordinate system. Although the members of this class are defined using the Cartesian coordinate system, components of the radial coordinate system (magnitude and angle) are member functions.

The vector can be manipulated by rotating and scaling.
