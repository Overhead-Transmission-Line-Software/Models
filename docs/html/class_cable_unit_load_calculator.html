Detailed Description

OVERVIEW

This class generates unit loads for a single transmission cable, subjected to a specific weather load case.

Unit loads are generated by modeling a cylinder (potentially subjected to radial icing) in a free steam flow.

ASSUMPTIONS
  • uniform loading (constat wind speed, equal ice accumulation)
  • drag coefficient is 1.0 (no aerodynamic effects)
