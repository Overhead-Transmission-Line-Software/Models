Detailed Description

CATENARY3D OVERVIEW

This class models a catenary in 3D.

This class is a wrapper for the 2D catenary, making its function applicable for modeling transmission cables by adjusting 2D catenary endpoint spacing for inclined spans with transverse loading.

COORDINATE SYSTEM

x = horizontal y = transverse z = vertical
